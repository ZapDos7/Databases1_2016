-- README FILE --
Διευκρινήσεις χρήσης της εφαρμογής/παραδοχές κατά την ανάπτυξη:

->Δεν έχουν παρθεί μέτρα για επιθέσεις SQL injection, διότι θεωρήθηκε πλεονασμός

->Στις φόρμες Presentation of Artists και Presentation Of Songs δεν είναι
απαραίτητη η συμπλήρωση κανενός πεδίου, ειδικά στην πρώτη απ'τις δύο δεν είναι
απαραίτητη ούτε η επιλογή Occupation

->Για να τρέξει η εφαρμογή απαιτείται χρήση browser που αναγνωρίζει HTML5

->Δεχόμαστε ότι μπορεί στη βάση να υπάρχουν καλλιτέχνες οι οποίοι δεν έχουν κανένα
απ'τα Occupations τραγουδιστής, συνθέτης, στιχουργός (δεν αναφέρεται προσδιορισμός
αυτού του χαρακτηριστικού κατά την εισαγωγή του καλλιτέχνη και υπάρχουν ήδη
εγγραφές που πληρούν την παραδοχή αυτή)

->Δεχόμαστε ότι όλα τα τραγούδια δεν υπάρχουν απαραίτητα σε CD, μιας και δεν
υπάρχει σχέση/table που να συνδέει αυστηρά όλα τα τραγούδια και τα CD στα
οποία ανήκουν (η τιμή CD ενός τραγουδιού είναι null-able)

->Για να τρέξει η εφαρμογή είναι απαραίτητη η χρήση Windows (δεν έχει δοκιμαστεί
σε περιβάλλον Linux), καθώς και η ύπαρξη της συγκεκριμένης βάσης δεδομένων στον
υπολογιστή ως server (με όνομα και κωδικό singer, port 3306, και port web διεπαφής
9090). Το app ακούει στην url διεύθυνση http://localhost:9090, μόνο αφού τρέχει το
αρχείο app.py στο shell της python

->Η ανάπτυξη έγινε σε python 2.7.11, με χρήση bottle 0.12.9, pymysql 0.7.2, html5,
css και mysql server 5.7
